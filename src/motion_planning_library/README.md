# Motion Planning 
Library of motion planning algorithms.

# Dependencies
1. CMake 
2. Eigen3
3. GTest

# Installing Dependencies
```
sudo apt install libeigen3-dev=3.3.7-2
sudo apt install libgtest-dev=1.10.0-2
sudo apt install cmake=3.16.3-1ubuntu1
```

# Building the Code
```
cmake -B build .
cmake --build build
```
